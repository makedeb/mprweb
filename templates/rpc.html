{{ "Package Search" | tr }}

Package searches can be performed by issuing HTTP GET requests of the form /rpc/?v=5&type=search&by=field&arg[]=keyword, where keywords is the search argument and field is one of the following values:

  • name: Search for packages that have a pkgname listing for keyword.
  • name-desc: Search for packages that have a pkgname or pkgdesc listing for keyword.
  • maintainer: Search for packages that have a maintainer listing for keyword.
  • depends: Search for packages that have a depends listing for keyword.
  • makedepends: Search for packages that have a makedepends listing for keyword.
  • optdepends: Search for packages that have a optdepends listing for keyword.
  • checkdepends: Search for packages that have a checkdepends listing for keyword.

{{ "Package Details" | tr }}

Package information can be obtained by issuing HTTP GET requests of the form /rpc/?v=5&type=info&arg=[]=pkg1&arg=pkg2&... where pkg1, pkg2... are the names of packages to retrieve package details for.

{{ "Examples" | tr }}

Search: /rpc/?v=5&type=search&arg=foobar

Search by maintainer: /rpc/?v=5&type=search&by=maintainer&arg=john

Search packages that have boost as a makedependslisting: /rpc/?v=5&type=search&by=makedepends&arg=boost

Get info of a package

: /rpc/?v=5&type=info&arg[]=foobar

Get info of multiple packages

: /rpc/?v=5&type=info&arg[]=foo&arg[]=bar
